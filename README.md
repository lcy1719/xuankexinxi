【技术栈】
①：架构: B/S、MVC
②：系统环境：Windows
③：开发环境：IDEA、JDK1.8、Maven、MySQL
④：技术栈：Java、Mysql、SpringBoot、Mybatis、HTML、拦截器

主要功能∶3个角色
1.学生的信息管理有以下功能：
学生信息管理，选课信息管理，考勤信息管理，请假信息管理，成绩信息管理
2.教师的信息管理有以下功能：
班级信息管理，教师信息管理，课程信息管理，选课信息管理，考勤信息管理，请假信息管理，成绩信息管理
3.管理员有以下功能：
班级信息管理，教师信息管理，课程信息管理，选课信息管理，考勤信息管理，请假信息管理，成绩信息管理
